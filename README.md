# DCGreener

너님의 디씨 생활을 그린-하게 만들어 드립니다.

전제조건
- 니가 고닉으로 싼 똥만 치울 수 있습니다.
- 한번에 다 안지워질 수도 있으니까 최소 3번 돌리세요.

사용방법
* JAR 실행
  * DCGreener.jar 파일을 다운로드한다.
  * 커맨드라인에서 jar 파일이 다운로드 된 경로로 이동한다.
  * java -jar DCGreener.jar 으로 실행한다.

* 소스 컴파일
  * IntelliJ 다운받고 이 프로젝트를 깃으로 연다.
  * 컴파일한다.
  * DCGreener 클래스를 실행한다.
